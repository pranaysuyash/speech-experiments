This system is a **Public Lab**. By uploading a file or starting a run, you agree to the following terms.

## 1. No Privacy
**ALL RUNS ARE PUBLIC.**
- Any file you upload is accessible to anyone with the system URL.
- Any transcript or artifact produced is public.
- **DO NOT** upload PII (Personally Identifiable Information), confidential meetings, or sensitive data.

## 2. No Ownership
- You do not own the runtime environment or the artifacts.
- You cannot "lock" or "claim" a run.
- Other builders may view, copy, or reference your run ID.

## 3. No Permanence
- **Volatile Storage**: The system makes no guarantee of data retention.
- **Deletion**: Runs may be deleted without warning to clear disk space or reset the state.
- **No Backup**: Do not use this system as a system of record. Download your artifacts immediately if you need them.
