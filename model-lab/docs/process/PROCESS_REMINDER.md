# Process Reminder

**Quick Reference for Agents:**

- **Start Here:** `prompts/workflow/agent-entrypoint-v1.0.md`
- **Ticket Discipline:** Always create/update `docs/WORKLOG_TICKETS.md` BEFORE implementing.
- **Evidence Types:** `Observed`, `Inferred`, `Unknown` — never upgrade `Inferred` without verification.
- **Preservation First:** Investigate before deleting; prefer activation or archiving.
- **Staging:** Use `git add -A` comprehensively.
- **No Branches:** Work on main unless user explicitly creates a branch.
- **Audit → Ticket:** Convert every actionable audit finding to a ticket immediately.
- **Prompts:** Use `prompts/README.md` to select the right prompt; record in artifacts.

If unsure, run `prompts/workflow/pre-flight-check-v1.0.md`.
