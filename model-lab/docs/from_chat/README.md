# Chat Captures Index (`docs/from_chat/`)

This folder stores **verbatim-ish** chat-provided research notes so we can trace claims back to their source text.

**Evidence convention**

- **Observed**: Reproduced inside Model‑Lab runs (`model-lab/runs/**`).
- **Reported**: Copied from external notes / web sources / chats; not reproduced here yet.
- **Unknown**: Not enough info to classify.
